Mock Welcome Message Test

What this tests

This page tests the data-webchat-mock-welcome attribute which injects a client-side welcome message without triggering the agent.

Expected behavior:

  • When you open the chat, a welcome message appears immediately
  • The message is injected client-side (no agent call)
  • No startConversation event is sent to the agent
  • Typing a message works normally and reaches the agent
  • Clicking restart shows the mock welcome again
